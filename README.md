# Royal Enfield Reviews

Detaljne recenzije Royal Enfield motora — stranica napravljena s HTML5 i CSS3.

## 📱 Opis projekta

Royal Enfield Reviews je **statička web stranica** posvećena recenzijama Royal Enfield motora. Stranica prikazuje različite modele motora podijeljene po kategorijama s detaljnim informacijama, specifikacijama, prednostima i manama.

Projekt je napravljan kao dio kolegija **"Uvod u Web tehnologije"** na UNIZD-u.

---

## 🏍️ Stranice projekta

1. **index.html** — Naslovnica s predstavljanjem Royal Enfielda
2. **starter-motori.html** — Hunter 350 i Meteor 350
3. **sport-touring.html** — Super Meteor 650 i Interceptor 650
4. **adventure.html** — Himalayan 450
5. **kontakt.html** — Kontakt forma

---

## ✨ Značajke

✅ **Semantički HTML5** — Koristi `<header>`, `<nav>`, `<main>`, `<footer>`  
✅ **Responsive dizajn** — Radi na mobilelu (375px) i desktopu (1280px)  
✅ **Hamburger izbornik** — Samo CSS, bez JavaScripta  
✅ **CSS varijable** — Tokeni za boje, razmake, fontove  
✅ **Pristupačnost** — Vidljiv fokus, skip link, alt za slike  
✅ **SEO optimiziran** — Meta tagovi, Open Graph, jedinstveni naslovi  
✅ **Forma s validacijom** — HTML validacija (required polja)  

---

## 🛠️ Tehnologije

- **HTML5** — Semantička struktura
- **CSS3** — Flexbox, Grid, Media queries, Transitions
- **CSS varijable** — Tokeni za konzistentnost
